# 民航客票销售订座系统需求摘要

资料来源：`~/Downloads/民航客票销售订座系统_需求文档.docx`。

## 项目定位

本系统是教学/实训用途的民航客票销售模拟系统。系统只使用测试数据，不连接真实航司、真实支付或真实票务系统。

## 功能范围

- 首页查询：出发机场/城市、到达机场/城市、三字码、出发日期、成人/儿童/婴儿人数。
- 机场基础数据：不少于 50 个机场/城市展示项，包含名称、三字码、地区、机场名称；当前覆盖用户指定国内主流与境外常用 IATA 三字码。
- 航班查询：航班号、机场、起降时间、时长、机型、客座率/余票、舱位、价格、航司标识；用户指定机场之间均可查到可订座模拟航班。
- 客票预订：成人、儿童、婴儿、无成人陪伴儿童（UM）、联系人、订单确认。
- 订单管理：订单列表、详情、打印订单/电子客票行程单、取消、退票、改签模拟。
- 运输规则：行李、退票、改签、特殊旅客运输规则。
- 电子客票：PNR、票号、旅客、航程、二维码区域、打印/保存 PDF 模拟。

## 优先级

- P0：首页查询、航班列表、成人票预订、订单确认。
- P1：儿童/婴儿票预订、行李/退票规则展示、电子客票。
- P2：无成人陪伴儿童票、订单管理（取消/改签/退票）。

## 非功能要求

- 蓝色主色，橙色强调色。
- PC、平板、手机响应式。
- 表单校验、错误提示、加载提示、关键操作二次确认。
- 支持 Web 访问，也支持 Windows 桌面应用打包。
- ToB 桌面授权：首次联网邀请码激活，绑定当前设备，后续离线校验签名授权文件；超过默认 30 天离线宽限期后联网校验服务端签名复核回执。
