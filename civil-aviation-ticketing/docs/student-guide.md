# 学生使用说明书

## 启动方式

### Web 模式

```bash
npm install
npm run dev
```

打开浏览器访问：`http://127.0.0.1:5173/`

### Windows 桌面模式

双击 `release/民航客票销售订座系统-1.0.3-x64.exe`。

## 订票练习流程

1. 在首页选择或输入出发/到达机场三字码、出发日期和旅客人数。
2. 点击“搜索航班”；输入框支持城市名、机场名和 IATA 三字码，例如 `PEK`、`PVG`、`LHR`。
3. 在航班列表中查看航班号、机场、起降时间、机型、客座率、余票和票价。
4. 选择舱位后填写乘机人信息。
5. 儿童/婴儿需关联成人旅客；UM 需填写送机人和接机人。
6. 检查订单确认区的航班、旅客和价格。
7. 点击“确认预订”，生成模拟订单。
8. 查看电子客票/行程单，可点击“打印/保存PDF”。
9. 进入订单管理，可练习取消、退票和改签模拟。

## 注意事项

- 所有航班、票号、二维码和价格均为教学模拟数据。
- 系统不连接真实航司，不产生真实订单。
- 退票和改签只用于学习规则和计算过程。
